# Laravel Inertia Comparison

This project targets Laravel-like Inertia ergonomics for Node.js, not a full Laravel clone.

## Included

- `res.inertia(...)`
- shared props
- flash messages
- validation redirect flow
- error bags
- auth hooks
- protected route helpers
- deferred props
- merge props
- SSR server support

## Not Included

- router framework
- database layer
- migrations
- password hashing choice
- OAuth provider
- queue system
- mail system
- Laravel Precognition
- history encryption

Node apps keep using Express routing and their preferred database/auth libraries.
